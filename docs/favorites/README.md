# Favourites Feature

The favourites feature allows users to bookmark their favourite coffee roasters for quick access. Favourites are displayed on a dedicated page and can be managed from both the roaster list and detail views.

## Main Capabilities
- **Add/Remove Favourite:** Users can add or remove roasters from their favourites with a single click.
- **Favourites Page:** View all favourited roasters in one place.
- **Persistence:** Favourites are stored in localStorage for guests and in the backend for authenticated users.
- **Sync:** When logged in, favourites sync with the backend.

## User Flow
1. User clicks the heart icon to add/remove a favourite.
2. Favourites are updated in real time and reflected in the UI.
3. The favourites page displays all saved roasters.
